    • Combination Lock Challenge (C/C++)

    The Challenge

    Implement a program to find all possible words of two letters or more created by a series of N lettered wheels with M letters per wheel, like a combination lock.

    Rules

    1. Words must be formed from a continuous forward run of letters, can start at any wheel, but cannot “wrap”. The maximum word length will therefore be M.
    2. There is no need to remove duplicate words and the order of the output word list does not matter.
    3. A textfile must be used to represent the wheels. The first two lines define the number of wheels and the number of letters per wheel. Each following line describes the letters in one wheel.
    4. A textfile must be used to represent the dictionary which contains an unsorted list of valid words with one word per line.
    5. The program should be written in C/C++ to run on Windows on the command line, taking wheels.txt and dictionary.txt as arguments.

    Running The Program

    To run the program, go to the correct folder location in the command line and type find_words.exe wheels.txt dictionary.txt.
